# Install

`@use-funnel` supports various routers. It supports the following routers:

- [react-router-dom](https://reactrouter.com/web/guides/quick-start)
- [next.js page router](https://nextjs.org/docs/pages)
  - [next.js app router](https://nextjs.org/docs/app-building) uses [browser history](https://developer.mozilla.org/en-US/docs/Web/API/History).
- [@react-navigation/native](https://reactnavigation.org/docs/getting-started)
- [browser history](https://developer.mozilla.org/en-US/docs/Web/API/History)

## Check React Version

| React | ^16.8 |
| :---: | :---: |

## Installation

Choose the router you want to use and install the package with the following command:

```shell npm2yarn
npm install @use-funnel/react-router-dom --save
npm install @use-funnel/next --save
npm install @use-funnel/react-navigation-native --save
npm install @use-funnel/browser --save
```

If you don't have the router you want or want to manage the history directly, see [Creating a Router](/docs/custom-router).
